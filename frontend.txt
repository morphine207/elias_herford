1. PreferencesPrompt.jsx

Collects user learning style and stores in React state or localStorage.

Fields:

Learning preference: visual, audio, text

Role description

Detail level: short, balanced, in-depth

News updates: on/off

const [preferences, setPreferences] = useState({
  style: "visual",
  role: "",
  detail: "balanced",
  news: true,
});

function handleSubmit() {
  localStorage.setItem("userPrefs", JSON.stringify(preferences));
  navigate("/course/1");
}

3. Slide.jsx

Renders one slide based on user preference.

Logic:

If visual: display image/video

If audio: play narration

If text: show written description

function Slide({ slide, preferences }) {
  if (preferences.style === "visual" && slide.image)
    return <img src={slide.image} alt="Slide Visual" />;
  if (preferences.style === "audio" && slide.audio)
    return <audio controls src={slide.audio} />;
  return <p>{slide.content}</p>;
}
